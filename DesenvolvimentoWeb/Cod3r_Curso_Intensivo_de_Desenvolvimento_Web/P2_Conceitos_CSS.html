----------------------------------------- BOX MODEL - Define o espaçamento entre os elementos do site Padding: Espaçamento entre o conteudo interior, e o conteudo externo Podendo se separar em 4: Padding-top: Define o espaçamento do topo Padding-right: Define o espaçamento da direita Padding-bottom: Define o espaçamento do fundo Padding-left: Define o espaçamento da esquerda Margin: Margem do elemento da paddind Podendo se separar em 4: Margin-top: Define o espaçamento do topo Margin-right: Define o espaçamento da direita Margin-bottom: Define o espaçamento do fundo Margin-left: Define o espaçamento da esquerda Exemplos: Colocando de uma vez: (Movimentando no sentindo horário) margin: 10px 20px 30px 40px; //Define a margem de todos os lados - margin: superior, direita, inferior, esquerda; Colocando separados: margin-top: 10px; //Define a margem superior margin-right: 20px; //Define a margem da direita margin-bottom: 30px; //Define a margem do fundo margin-left: 40px; //Define a margem da esquerda
